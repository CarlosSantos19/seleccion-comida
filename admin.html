Descargar Backup

Descarga una copia de seguridad de todos los datos actuales en formato JSON. Guarda este archivo en un lugar seguro.

Restaurar Base de Datos

Restaura los datos desde un archivo de backup JSON previamente descargado. Esta acción sobrescribirá todos los datos actuales.

Advertencia: La restauración reemplazará todos los datos actuales en la base de datos. Asegúrate de tener un backup reciente antes de proceder.

Limpiar Base de Datos

Elimina TODOS los datos de la base de datos. Esta acción es irreversible. Asegúrate de tener un backup antes de continuar.

Operación completada exitosamente.

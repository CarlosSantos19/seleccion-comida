# Aplicación de Selección de Comida

Una aplicación web moderna e intuitiva para que los funcionarios seleccionen sus alimentos preferidos.

## Características

- **Interfaz atractiva**: Diseño moderno con gradientes, animaciones y efectos visuales
- **Contadores interactivos**: Selecciona fácilmente la cantidad de cada alimento con botones + y -
- **Persistencia de datos**: Las selecciones se guardan automáticamente en el navegador
- **Resumen completo**: Visualiza todas las selecciones en una tabla organizada
- **Exportación a Excel**: Descarga los datos en formato CSV
- **Responsive**: Funciona perfectamente en dispositivos móviles, tablets y escritorio

## Uso

1. Abre el archivo `index.html` en tu navegador
2. Selecciona un funcionario de la lista desplegable
3. Usa los botones + y - para seleccionar la cantidad de cada alimento
4. Haz clic en "Guardar Selección"
5. Visualiza el resumen de todas las selecciones en la tabla
6. Exporta los datos a Excel si lo necesitas

## Tecnologías

- HTML5
- CSS3 (con animaciones y efectos avanzados)
- JavaScript (Vanilla)
- Font Awesome Icons
- LocalStorage para persistencia de datos

## Instalación

No requiere instalación. Solo descarga los archivos y abre `index.html` en tu navegador.

## Archivos del Proyecto

- `index.html` - Estructura principal de la aplicación

- `app.js` - Lógica de la aplicación
